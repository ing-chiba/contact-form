## このコースで学ぶこと

フロントエンドで使用する JavaScript の現代的な書き方（ES6）を学びます。
React や TypeScript の前提知識になります。

| 概念 | 説明 |
|------|------|
| アロー関数 | `const fn = () => {}` の形で書く関数 |
| 分割代入 | 配列・オブジェクトを簡潔に展開する構文 |
| スプレッド演算子 | 配列・オブジェクトをコピー・結合する構文 |
| Promise / async・await | 非同期処理を扱う仕組み |
