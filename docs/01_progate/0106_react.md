## このコースで学ぶこと

フロントエンドのフレームワーク React の基礎を学びます。
このプロジェクトのフロントエンドは React で実装します。

| 概念 | 説明 |
|------|------|
| コンポーネント | UI を部品に分けて再利用する仕組み |
| Props | 親から子へデータを渡す仕組み |
| State | コンポーネント内で変化するデータを管理する仕組み |
| イベントハンドラ | クリックなどのユーザー操作に反応する処理 |
