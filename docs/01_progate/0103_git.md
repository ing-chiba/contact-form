## このコースで学ぶこと

コードの変更履歴を管理する Git の基本を学びます。
次のステップで実際に Git を使うので、ここで概念を理解しておきましょう。

| 概念 | 説明 |
|------|------|
| リポジトリ | コードの変更履歴を管理する場所 |
| コミット | 変更を記録するスナップショット |
| ブランチ | 独立した作業ラインを作る仕組み |
| マージ | ブランチを統合する操作 |
