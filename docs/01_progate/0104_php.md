## このコースで学ぶこと

Laravel は PHP で動いているため、言語の基本を押さえておきましょう。

| 概念 | 説明 |
|------|------|
| 変数 | データを入れる箱 |
| 配列 | 複数のデータをまとめる仕組み |
| 関数 | 処理をまとめて再利用できるようにする仕組み |
| クラス | データと処理をひとまとめにしたテンプレート |
