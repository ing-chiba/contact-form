## このコースで学ぶこと

ターミナル（コマンドライン）の基本的な操作を学びます。
開発ではコマンドを使う場面が多いため、最初に慣れておきましょう。

| コマンド | 説明 |
|----------|------|
| `ls` | ファイル一覧を表示する |
| `cd` | ディレクトリを移動する |
| `mkdir` | ディレクトリを作成する |
| `touch` | ファイルを作成する |
