## このコースで学ぶこと

JavaScript に型を追加した TypeScript を学びます。
このプロジェクトのフロントエンドは TypeScript で実装します。

| 概念 | 説明 |
|------|------|
| 型アノテーション | 変数や引数に型を明示する書き方 |
| インターフェース | オブジェクトの型を定義する仕組み |
| 型推論 | 型を書かなくても TypeScript が自動で推測してくれる機能 |
| ジェネリクス | 型を引数のように渡せる仕組み |
