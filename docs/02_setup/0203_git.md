# 0203 Git セットアップ

## 1. Git の初期設定

Git にユーザー情報を登録します。コミット履歴に表示される名前とメールアドレスです。

```bash
git config --global user.name "あなたの名前"
git config --global user.email "あなたのメールアドレス"
```

設定を確認してください。

```bash
git config --list
```
